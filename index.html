Privacy Policy

Yaoji - Obsidian Daily Todo Sync

Effective Date: March 12, 2026

This privacy policy describes how the Yaoji browser extension ("the Extension") handles user data.

1. Overview

Yaoji is a Chrome extension that syncs daily journal todo items between your browser and a locally running Obsidian vault. The Extension communicates with the Obsidian Local REST API plugin on your own computer (127.0.0.1). Your todo data never leaves your local machine through the Extension's core functionality.

2. Information We Collect

2a. Data stored locally

The Extension stores the following in your browser's local storage (chrome.storage.local):

  • Configuration settings: Obsidian connection details (protocol, API key), note folder path, date format, UI preferences (theme, auto-refresh), Pomodoro timer settings, weather city name, and feature toggles.
  • Membership session: If you use the optional membership feature, your login session token, device identifier, and membership status are stored locally.

This data never leaves your browser except as described below.

2b. Data sent to external services

The Extension communicates with the following external services:

1) Obsidian Local REST API (127.0.0.1 ports 27123/27124)

Purpose: Read and write daily journal files in your Obsidian vault.
Data sent: File read/write requests containing todo text.
This is local-only communication on your own machine.

2) Hitokoto API (v1.hitokoto.cn)

Purpose: Fetch a random inspirational quote for the new tab page.
Data sent: No user data. Read-only GET request.

3) Amap Weather API (restapi.amap.com)

Purpose: Display weather information on the new tab page.
Data sent: The city name you configured in settings. No personal identifiers.

4) Bark Push Service (api.day.app)

Purpose: Send Pomodoro timer notifications to your mobile device.
Data sent: Notification title and body text.
Only used if you explicitly enable and configure it in settings.

5) Supabase Backend

Purpose: Optional membership system for premium features.
Data sent: Email address (for registration/login), device identifier, authentication tokens.
Only used if you actively use the membership feature.

6) OpenAI-compatible API (user-configured endpoint)

Purpose: Optional OCR image text recognition.
Data sent: Image data for text recognition.
Only used if you enable OCR and configure your own API endpoint and key in settings.

3. How We Use Information

All collected information is used solely to provide the Extension's stated functionality:

  • Todo synchronization with your local Obsidian vault
  • Weather display on the new tab page
  • Pomodoro timer notifications
  • Optional membership management
  • Optional OCR text recognition

4. Information Sharing

We do not sell, trade, or transfer your personal information to third parties. Data is only sent to the specific services listed above, each for its stated purpose.

5. Data Storage and Security

  • Core data (todos, settings) is stored locally in your browser and your Obsidian vault on your own machine.
  • Membership data (if used) is stored in a Supabase-hosted database with standard security practices.
  • API keys and sensitive configuration values are stored in chrome.storage.local and are not accessible to websites you visit.
  • The Extension sanitizes file paths and validates configuration to prevent path traversal and injection attacks.

6. Data Retention

  • Local browser storage: Data persists until you uninstall the Extension or clear browser data.
  • Membership data: Retained on the backend while your account is active. Contact us to request deletion.

7. Your Rights and Choices

  • You can view and modify all stored settings through the Extension's options page.
  • You can export and import your configuration.
  • You can disable optional features (weather, Bark notifications, inspiration capture, OCR, membership) at any time.
  • You can uninstall the Extension to remove all locally stored data.
  • To delete membership account data, contact us at the email below.

8. Children's Privacy

This Extension is not directed at children under 13. We do not knowingly collect personal information from children.
