曜记 - Obsidian 待办同步

隐私政策 | 最后更新：2026-03-12

概述

曜记（以下简称"本扩展"）是一款 Chrome 浏览器扩展，用于在浏览器中同步和管理 Obsidian 日记的待办事项。本隐私政策说明本扩展访问哪些数据、如何使用、以及如何保护这些数据。

本地存储的数据

本扩展使用 Chrome 存储 API（chrome.storage.local）在用户设备上保存以下数据：

  • Obsidian API 连接设置（协议选择和 API 密钥）
  • 笔记文件配置（文件夹路径、文件名格式、章节标题）
  • 界面偏好（主题、天气城市、自动刷新、背景图场景）
  • 番茄钟设置和会话状态
  • 目标墙配置
  • 灵感捕获偏好
  • OCR API 配置（端点地址、模型名称、提示词）
  • 会员登录令牌（如已登录）

这些数据不会离开用户设备，除以下情况外。

与本地 Obsidian 的数据通信

本扩展通过 Obsidian Local REST API 插件与运行在用户本地的 Obsidian 应用通信（地址为 127.0.0.1），读写 Obsidian Vault 中的 Markdown 文件。此通信完全在用户的本地网络内进行，不经过互联网。

与外部服务的数据通信

本扩展向以下外部服务发送网络请求：

  1. 一言 API（v1.hitokoto.cn）
    获取随机金句显示在新标签页上。仅发送 GET 请求获取金句内容，不传送任何用户数据。
  2. 高德天气 API（restapi.amap.com）
    使用用户在设置中配置的城市名称查询天气信息。仅传送城市名称，不包含位置数据、IP 地址或其他个人信息。
  3. Bark 推送 API（api.day.app）
    用户启用 Bark 推送通知后，番茄钟完成时向 Bark API 发送通知消息。此功能默认关闭，仅在用户主动配置启用后才会调用。
  4. Supabase
    用户使用会员功能时，本扩展与 Supabase 后端通信，处理邮箱验证码登录、会员状态查询、激活码兑换和支付订单管理。
  5. 用户配置的 OCR API
    用户启用图像文字识别功能后，本扩展将图片发送到用户配置的 API 端点。本扩展会在访问自定义 API 域名前征求用户的明确许可。

从网页访问的数据

用户使用内容脚本功能时，本扩展仅在用户明确操作后访问以下页面数据：

  • 选中文本：用户通过右键菜单或灵感捕获功能时读取。
  • 页面标题和 URL：用户选择附加来源信息时读取。
  • 图片 URL：用户通过右键菜单操作图片时获取。

这些数据仅发送到用户本地的 Obsidian Vault 或用户配置的 OCR API 端点，不会被收集、汇总或传输给任何其他方。

数据共享

本扩展不会出售、交易或将用户数据转让给任何第三方。数据仅与用户本地的 Obsidian 应用（通过本地 API）及上述列出的外部服务（仅在用户主动启用后）共享。

数据安全

  • 与本地 Obsidian Vault 的通信通过本地回环地址进行，支持 HTTPS 加密模式。
  • API 密钥存储在 Chrome 扩展存储中，不会传输到其目标端点以外的服务器。
  • 会员系统使用 Supabase 内置的安全机制。
  • 内容脚本运行在 Shadow DOM 中，与页面脚本隔离。

远程代码

本扩展不加载或执行任何远程托管的代码。所有 JavaScript 均从扩展包内的文件运行。第三方库均作为本地副本包含在扩展包中。

儿童隐私

本扩展不面向 13 岁以下儿童。我们不会故意收集儿童的个人信息。

用户选择

  • 可在设置页面独立关闭任何外部服务集成。
  • 可导出、导入或重置所有扩展配置。
  • 卸载扩展会删除所有本地存储的数据。
  • 可随时通过 Chrome 扩展设置撤销可选的主机权限。

隐私政策变更

我们可能会不定期更新本隐私政策。变更将通过更新本页顶部的"最后更新"日期来通知。
